# Testovací checklist

Po nahrání na GitHub Pages ověřte:

- Aplikace se spustí.
- Logo a favicon se načtou.
- Grafický návrh se vykreslí.
- Změna rozměrů střechy přepočítá návrh.
- Tabulka překážek funguje.
- Kliknutí na tašku vyplní kontrolu tašky.
- Kliknutí na překážku vyplní kontrolu detailu.
- Nápověda se otevře.
- CSV export funguje.
- Tisk/PDF funguje přes tisk prohlížeče.
